# 概述
## 核心功能

android app,通过无障碍和悬浮按钮，快速在微信等聊天内容截图。通过ocr识别文字，再通过文本位置在屏幕左侧还是右侧判断是对方说的话还是自己说的话，通过deepseek的api联网查询3条风格不同的话术，悬浮返回推荐话术。

## 其他功能

1. 配置deepseek的链接，模型， API key
2. 权限检查与快速配置
3. 提示词自定义。

# 核心方案
1.开启无障碍与悬浮按钮
2.点击悬浮按钮，对界面进行截图
3.ocr识别文字，通过文本在屏幕左侧还是右侧判断是对方说的话还是自己说的话，用"对方:"和"我:"标识
5.联网请求 DeepSeek，查询推荐回复。
6.删除对话截图

# 开发要求
1.使用java作为核心编程语言，可使用其他语言辅助
2.研发环境为中国境内，gradle等环境尽量使用国内镜像

3.生成readme文档

4.方案不合理的地方可以修改

# 其他要求

基础提示词类似：

```
你是一个聊天回复助手。根据用户提供的聊天上下文，生成3条回复建议。
每条回复需要标注风格类型：
1. 【保守】- 安全、礼貌、不容易出错的回复
2. 【激进】- 积极推进关系或话题的回复
3. 【出其不意】- 有趣、幽默或创意的回复
```

和获取的对话内容共同组成提示词，可根据返回结果对3个选项进行拆分，分条在悬浮窗展示，点击某条话术，自动复制到剪贴板
